Downloading the IMS Scientific Legacy data

The complete IMS Scientific Legacy Dataset, from which the HTML in this website was scripted, is available both in a plain text format and in JSON format. The source file for this bibliographic dataset is the plain text version, from which both the HTML for this website and the JSON export were scripted, using Python code developed by Jim Pitman for this purpose. The JSON file is formatted largely following the conventions of BibJSON. The python code is an extension of the BibServer software developed by Jim Pitman.

Assistance in publishing, documenting, and extending the BibServer code on GitHub has been provided by Benoit Dherin. We welcome further participation by software developers and data scientists interested in management, analysis and display of bibliographic data.

As the IMS Legacy Dataset is Open Data, you are free to use it, analyze it and offer displays of it however you wish, without any obligation to MSP or IMS. Still, if you make interesting or useful developments of this kind, we would appreciate it if you acknowledge this website as the source of the data, and if provide us pointers to whatever displays or services you may be able to create using the IMS Scientific Legacy Collection. We encourage you to find ways to make such efforts enhance the both the quantity and quality of public domain bibliographic data, and to contribute to Open Bibliography for Science, Technology, and Medicine.
